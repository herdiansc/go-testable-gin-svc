# go-testable-gin-svc

This is a repository of a web api framework built with go and gin-gonic framework.

## Goals
- Learn to build a web api from scratch with go and gin-gonic framework
- Learn to unit test

## Features
- Get endpoint
- Post endpoint
- Custom validation with go-playground/validator.10

## Endpoints
- GET /ping
- POST /divide
- POST /calculate
